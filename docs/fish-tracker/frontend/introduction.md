# Frontend Overview

The **CosmosMC Fish Tracker** frontend is a web app built with **Next.js** and **TypeScript** that serves as the user interface for tracking and visualizing fish/crab catches on the [CosmosMC](https://cosmosmc.org) Minecraft server.

## Documentation Structure

This frontend documentation is organized into focused sections:

- **[Architecture](architecture.md)** - Core technologies, application structure, and technical implementation
- **[Development](development.md)** - Setup instructions, build process, and development workflow

## Core Technologies

- **Next.js 16**: React framework with server-side rendering
- **Tailwind CSS**: Utility-first CSS framework
- **Framer Motion**: Animation library
- **React Icons**: Icon library

## Key Features

- **User Authentication**: Cookie-based login with persistent sessions
- **Fish & Crab Tracking**: Multi-gamemode support with rarities mapping
- **Statistics Dashboard**: Sorting, filtering, and visualization
- **Admin Panel**: User and data management
- **Demo Mode**: Demo with no required registration


## API Integration

- RESTful API communication with [backend](../backend) services
- TypeScript interfaces for type-safe API interactions
- Error handling and loading states
- Real-time data updates and authentication
