# Getting Started with Fish Tracker

## Request Access

To get started with Fish Tracker, you'll need an API key and login credentials. To obtain these join the [Discord server](https://discord.gg/Uah2dNRhFV) and navigate to the `fish-tracker-apps` channel. Submit your application and the team will review your request. Then you will be provided with your API key, username, and password.

## Access Information

- **Web App**: Available at [https://tracker.petarmc.com](https://tracker.petarmc.com)
- **Mod**: See [Mod Installation](mod/installation.md) for download links and setup
- **API**: Public endpoints documented [here](backend/api/public)

## Security Features

- All data sending uses Fernet encryption (AES 128-bit)
- User authentication with secure sessions
- Server-side decryption ensures data privacy

## Fish Rarity System

The system tracks fish by rarity tiers from Bronze to Mythical. See the [Fish Rarity Tiers](rarity-tiers.md) for complete details.

## Supported Gamemodes

- OneBlock
- Earth
- Survival
- Factions
- BoxSMP
