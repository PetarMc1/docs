# Installation

Follow these steps to install FishTracker on your Minecraft client.

## Required Dependencies

Before installing FishTracker, ensure you have the following mods installed:

- [Fabric API](https://modrinth.com/mod/fabric-api)
- [Cloth Config](https://modrinth.com/mod/cloth-config)
- [PetarLib](https://modrinth.com/mod/petarlib)
- [Mod Menu](https://modrinth.com/mod/modmenu) (optional)

## Installation Steps

:::tip
You can download development builds for each commit from the [CI](https://ci.petarmc.com/job/Fish%20Tracker%20Mod/)
:::


1. Download the latest FishTracker jar file from [Modrinth](https://modrinth.com/mod/fish-tracker-mod) or the [Releases](https://github.com/PetarMc1/fish-tracker-mod/releases) page.
2. Locate your Minecraft mods folder:
   - Windows: `%APPDATA%\.minecraft\mods`
   - macOS: `~/Library/Application Support/minecraft/mods`
   - Linux: `~/.minecraft/mods`
3. Place the downloaded jar file into the mods folder.
4. Launch Minecraft
