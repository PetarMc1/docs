# Dependencies

FishTracker relies on several libraries and mods to function. This page lists all dependencies and their purposes.

## Required Mod Dependencies

These mods must be installed alongside FishTracker:

### Fabric API
- **Purpose**: Core Fabric modding API providing hooks and utilities.
- **Download**: [Modrinth](https://modrinth.com/mod/fabric-api) or [CurseForge](https://www.curseforge.com/minecraft/mc-mods/fabric-api)

### Cloth Config API
- **Purpose**: Provides the configuration GUI framework used by FishTracker.
- **Download**: [Modrinth](https://modrinth.com/mod/cloth-config) or [CurseForge](https://www.curseforge.com/minecraft/mc-mods/cloth-config)

### PetarLib
- **Purpose**: Custom utility library providing additional functionality.
- **Download**: [Modrinth](https://modrinth.com/mod/petarlib) or [GitHub Releases](https://github.com/PetarMc1/petarlib/releases)



## Installing Dependencies

All required mods should be placed in your `.minecraft/mods` folder alongside FishTracker.

:::tip
Always download mods from official sources to avoid security risks.
:::

## Updating Dependencies

When updating FishTracker, check for updated dependency versions in the changelog. Update all dependencies simultaneously to avoid compatibility issues.
