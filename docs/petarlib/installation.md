# Installation

## Downloading and Installing

1. Download the latest version of PetarLib from [Modrinth](https://modrinth.com/mod/petarlib) or the [releases page](https://github.com/Petar_mc/PetarLib/releases)
2. Locate your Minecraft installation folder.
3. Place the downloaded `.jar` file into the `mods` folder.
4. Launch Minecraft with Fabric.
